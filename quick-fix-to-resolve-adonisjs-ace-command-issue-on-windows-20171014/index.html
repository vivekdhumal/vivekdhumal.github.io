Quick Fix to Resolve AdonisJs Ace Command Issue On Windows

October 14, 2017

Adonis Js is a MVC Framework for Node.js to write webapps with less code, in first impression I like it very much because it exactly like Laravel framework but in javascript.

I love the simplicity of it and yes it is really easy and fun to build web application in node with it, it has the ace command to generate controllers, models instead of manualy creating them, just like the laravel has the artisan command.

But when you first time run the ace command from your terminal on windows it will not work, it will probably tells you that 'ace' is not recognized as an internal or external command, operable program or batch file.. well adonis has nothing to do with this, its a windows, it does not recognise this ace file (at the root of your adonis app) as a executable program.

Well there are two simple methods to fix this issue, either you run node ace or you can create a simple batch file to the root of your app as follows

@ECHO  OFF
setlocal DISABLEDELAYEDEXPANSION
SET BIN_TARGET=%~dp0/ace
node "%BIN_TARGET%" %*

now you can run ace command directly without using node at the start and you can even store this batch file somewhere in bin directory and append that bin directory path to windows environment variables so that you don't required to create this batch file every time for every adonis app you create.

So go ahead and give it a shot, thank you.
