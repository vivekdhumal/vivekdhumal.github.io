How to make Vim Editor to look cool

August 12, 2017

Vim editor is look very 90's and its very boring to work with the old UI, as you may know that vim editor is the UNIX style editor and it is popular among the developer fraternity.

It is highly configurable text editor, rock stable and is continuously being developed to become even better.

If you look at the default vim UI it will look like as below.


Lets work on UI to make it more fancy and pleasant for the eyes, because, as a today's developer you need to enjoy the development. Well I am using Vim on windows, so if you are a mac user you can use MacVim.

I assume that you may know the basic vim commands like :w, :e, :q, :pwd and the different modes like insert, visual, command etc.

So in nutshell we are going to

  • enable syntax highlighting
  • enable the line numbering
  • change the color scheme
  • change the font size and font family
  • change the default line spacing to make it more cleaner.

First thing first, will start by creating a _vimrc file to our users directory, for mac user they need to create a .vimrc file to their root (~/) directory.

For your information _vimrc or .vimrc is vim configuration file, which helps you to make vim as per your preferences.

So will run the following command in vim to create a blank _vimrc file to our user directory.

:e C:/users/Vivek/_vimrc

Before writing anything to it, you need to download Fira code font which I personally like it and use it frequently and very beautiful Vim Atom Dark color scheme.

You can install the font to your machine and for color scheme, extract the vim-atom-dark-master.zip, copy the files from /colors directory into your "vim" directory well mine is in C:\Program Files (x86)\Vim\vim80\colors.

After that will come back to our _vimrc file and write the following code into it by pressing "i".

syntax enable                       "enable the syntax highlighting.
colorscheme atom-dark               "change to color scheme.
set guifont=Fira_Code:h10           "Set the default font family and size.
set number                          "Let's activate line numbers.
set linespace=15                    "set Line height.

Will just save it with :w and source it by :so % which means we are applying configuration globally.

And we are done, and if you can see any php file from your project or even _vimrc file you'll get to see the difference in UI theme changes like so.
