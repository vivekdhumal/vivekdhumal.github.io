Make symfony based console application

June 23, 2017

Symfony is a great framework in PHP where you can use its components individually in your normal projects where you don't required a bulky framework with a tons of libraries.

Now we are going to use one of its component called the console component, and we were creating a simple hello world application in it.

At this stage I'm assuming that you have a basic knowledge of Composer - PHP dependencies manager and its auto loading feature.

Okay then, lets create a sample project folder called symfony-console-app in that project create a blank composer.json file and then run

$ composer require symfony/console

After that your composer.json file will look like as follows:

{
    "require": {
        "symfony/console": "^3.3"
    }
}

Also it will create a vendor folder in root of the app which contains all the necessary dependencies.

Cool, now lets create a src folder to the root and create HelloCommand.php file and add the below code into that.

<?php

use Symfony\Component\Console\Command\Command;
use Symfony\Component\Console\Input\InputInterface;
use Symfony\Component\Console\Output\OutputInterface;

class HelloCommand extends Command
{
    public function configure()
    {
        $this->setName('hello')
            ->setDescription('It will print hello world on the console');
    }

    public function execute(InputInterface $input, OutputInterface $output)
    {
        $output->Writeln("Hello World!");
    }
}

first will import the require namespaces and then will extend our class with the symfony contract/interface called Command.

Then will implement the methods defined by the Command contract such as configure() and execute()

Inside the configure() method you can define your command name and its description about what it will do exactly in simple words.

The execute() method accept the two parameter of type InputInterface and OutputInterface respectively, where input interface object will help us to accept any arguments from command line and the output interface is allow you to just print the final output to the terminal.

Note - Here we are not accepting any arguments from command line so we are not using the InputInterface object but we will required to import it compulsory, just to maintain the method signature defined inside the contract

so we did write our first command class but we now going to learn how to make it works from commandline.

but before that we need to tell composer to autoload this class so that will change our composer.json code lil bit as follows:

{
    "require": {
        "symfony/console": "^3.3"
    },
    "autoload": {
        "psr-4": {
            "" : "src/"
        }
    }
}

here we are using psr-4 standard and we are telling composer to autoload files from src folder, after that we need to run the following command:

$ composer dump-autoload

This will refresh the composer autoload file.

Okay, lets create a file to the root without any extension called demo you can name it whatever you want and add the following code:

#!/usr/bin/env php
<?php

require __DIR__.'/vendor/autoload.php';

use Symfony\Component\Console\Application;

$app = new Application();

// register command
$app->add(new HelloCommand());

$app->run();

on the first line we are defining our environment as php, after that we are requiring the autoload.php file from vendor directory.

then import the symfon's application namespace

after that create the application instance and register that command as shown above and call the run method to boot the symfony app instance.

Tadda, now your are done with code now just execute it by using following steps:

$ php demo

it will list all the information about the command as bellow

if you run our hello command as below

$ php demo hello

the output will be as follows

$ Hello World!

Thats all folks, this is very basic application but you can make advance application using its full power.

No your job is to figure it out how you can accept the argument from the command line and print the output based on argument.
